[Ada 2012 Examples][1]
======================

A collection of examples of how Ada 2012 can be used.


Build dependencies
------------------

+ Bash
+ GNU Parallel (or `xargs`)
+ GNAT
+ Mercurial (`hg`)
+ Perl


Testing
-------

    make test


Building
--------

    make
